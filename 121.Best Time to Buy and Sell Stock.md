## Solution 1: accepted 3ms

DP.  
Time: O(n)  
Space: O(1)  

```java
public class Solution {
    public int maxProfit(int[] prices) {
        if (prices.length < 2) return 0;
        int max = 0;
        for (int i = 1; i < prices.length; i++) {
            if (prices[i] > prices[i-1]) {
                max = Math.max(max, prices[i] - prices[i-1]);
                prices[i] = prices[i-1];
            }   
        }
        return max;
    }
}
```
