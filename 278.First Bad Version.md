## Solution 1: accepted 18ms

Binary search.  

Feel the differences and commonalities. The first solution is recommended, as it is a wild card solution.  

```java
public int firstBadVersion(int n) {
    int lo = 1;
    int hi = n;
    while (lo + 1 < hi) {
        int mid = lo + (hi - lo) / 2;
        if (isBadVersion(mid)) {
            hi = mid;
        } else {
            lo = mid;
        }
    }
    return isBadVersion(lo)? lo : hi;
}
```
