## Solution 1:

```java
public class Solution {
    public int lengthOfLIS(int[] nums) {
        
        int len = nums.length;
        if (len < 2) return len;
        
        HashMap<Integer, Integer> map = new HashMap<>();
        for(int i = 0; i < len; i++) {
            map.put(i, 1);
        }
        
        int max = 1;
        for (int i = 1; i < len; i++) {
            for (int j = 0; j < len; j++) {
                if (nums[j] < nums[i]) {
                    int bigger = map.get(i) > map.get(j) + 1 ? map.get(i) : map.get(j) + 1;
                    map.put(i, bigger);
                }
            }
            max = max > map.get(i)? max :  map.get(i);
        }
        return max;
    }
}
```
