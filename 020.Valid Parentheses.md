## Solution 1

Related questions: 022, 032, 301.  
Priority: == > && > ||    

Time complexity: O(n)  
Space complexity: O(n)  


```java
class Solution {
    public boolean isValid(String s) {
        Stack<Character> stack = new Stack<>();
        for (int i = 0; i < s.length(); i++) {
            char cur = s.charAt(i);
            if (!stack.empty() &&
                (cur == ')' && stack.peek() == '(' ||
                 cur == ']' && stack.peek() == '[' ||
                 cur == '}' && stack.peek() == '{')) {
                stack.pop();
            }
            else {
                stack.push(cur);
            }
        }
        return stack.empty();
    }
}
```
