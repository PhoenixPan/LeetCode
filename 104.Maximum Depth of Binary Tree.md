## Solution 1: accepted 1ms

DFS.  

```java
public int maxDepth(TreeNode root) {
    if (root == null)
        return 0;
    int leftDepth = maxDepth(root.left) + 1;
    int rightDepth = maxDepth(root.right) + 1;
    return Math.max(leftDepth, rightDepth);
}
```
