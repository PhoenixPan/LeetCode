## Solution 1

Recursive. If we make change on t1 directly instead of making a new tree, it would be faster.  

Time complexity: O(n)  
Space complexity: O(n)  

```java
class Solution {
    public TreeNode mergeTrees(TreeNode t1, TreeNode t2) {
        if (t1 == null && t2 == null) 
            return null;
        if (t1 == null)
            return t2;
        if (t2 == null)
            return t1;
        
        TreeNode root = new TreeNode(t1.val + t2.val);
        root.left = mergeTrees(t1.left, t2.left);
        root.right = mergeTrees(t1.right, t2.right);
        return root;
    }
}
```

## Solution 2

DFS.  

Time complexity: O(n)  
Space complexity: O(n)  

```java
class Solution {
    public TreeNode mergeTrees(TreeNode t1, TreeNode t2) {
        if (t1 == null && t2 == null) 
            return null;
        if (t1 == null)
            return t2;
        if (t2 == null)
            return t1;
        
        Stack<TreeNode[]> stack = new Stack<>();
        stack.push(new TreeNode[] {t1, t2});
        
        while(!stack.isEmpty()) {
            TreeNode[] nodes = stack.pop();
            
            if (nodes[1] == null) {
                continue;
            }

            nodes[0].val += nodes[1].val;
            if (nodes[0].left == null) { // left or right order doesn't matter
                nodes[0].left = nodes[1].left;
            } else {
                stack.push(new TreeNode[] {nodes[0].left, nodes[1].left});
            }

            if (nodes[0].right == null) {
                nodes[0].right = nodes[1].right;
            } else {
                stack.push(new TreeNode[] {nodes[0].right, nodes[1].right});
            }
        }
        return t1;
    }
}
```

## Solution 3

BFS.  

Time complexity: O(n)  
Space complexity: O(n)  

```java
class Solution {
    public TreeNode mergeTrees(TreeNode t1, TreeNode t2) {
        if (t1 == null && t2 == null) 
            return null;
        if (t1 == null)
            return t2;
        if (t2 == null)
            return t1;
        
        Queue<TreeNode[]> queue = new LinkedList<>();
        queue.add(new TreeNode[] {t1, t2});
        
        while(!queue.isEmpty()) {
            int size = queue.size();
            
            for (int i = 0; i < size; i++) {
                TreeNode[] nodes = queue.poll();
                if (nodes[1] == null) {
                    continue;
                }

                nodes[0].val += nodes[1].val;
                if (nodes[0].left == null) {
                    nodes[0].left = nodes[1].left;
                } else {
                    queue.add(new TreeNode[] {nodes[0].left, nodes[1].left});
                }

                if (nodes[0].right == null) {
                    nodes[0].right = nodes[1].right;
                } else {
                    queue.add(new TreeNode[] {nodes[0].right, nodes[1].right});
                }
            }
        }
        return t1;
    }
}
```
