## Solution 1

Naive solution

Time complexity: O(n)  
Space complexity: O(n)

```java
class Solution {
    HashMap<Integer, Integer> map;

    public int[] findMode(TreeNode root) {
        map = new HashMap<>();
        helper(root);

        int max = 0;
        ArrayList<Integer> list = new ArrayList<>();

        for (Map.Entry<Integer, Integer> entry : map.entrySet()) {
            if (entry.getValue() < max) {
                continue;
            }
            if (entry.getValue() > max) {
                list.clear();
            }
            list.add(entry.getKey());
            max = entry.getValue();
        }

        int[] result = new int[list.size()];
        for (int i = 0; i < list.size(); i++) {
            result[i] = list.get(i);
        }

        return result;
    }

    private void helper(TreeNode node) {
        if (node == null)
            return;
        map.put(node.val, map.getOrDefault(node.val, 0) + 1);
        helper(node.left);
        helper(node.right);
    }
}
```


## Solution 2

O(1) extra space solution.

Time complexity: O(n)  
Space complexity: O(1)

```java
class Solution {
    ArrayList<Integer> list;
    Integer pre;
    int cur = 0;
    int max = 0;
    public int[] findMode(TreeNode root) {
        list = new ArrayList<>();
        traverse(root);
        int[] result = new int[list.size()];
        for (int i = 0; i < list.size(); i++) {
            result[i] = list.get(i);
        }
        return result;
    }
    
    private void traverse(TreeNode node) {
        if (node == null) return;
        
        traverse(node.left);
        
        if (pre != null && node.val == pre) {
            cur++;
        } 
        else {
            cur = 1;    
        }
        
        if (cur == max) {
            list.add(node.val);
        }
        else if (cur > max) {
            list.clear();
            list.add(node.val);
            max = cur;
        }
        pre = node.val;
        
        traverse(node.right);
    }
}         
```
